📚 Content Management Guide

This guide explains how to update club information and images without coding knowledge.

Managing Club Information

All club information is stored in a single JSON file that you can edit with any text editor.

  1. Locate the clubs file:
    Open the file at /data/clubs.json in any text editor (Notepad, TextEdit, VS Code, etc.)
  2. Find the club to edit:
    Each club is an object inside square brackets. Look for the club by its "name" field.
  3. Edit the information:
    Update any field you need. Common fields include:
    • name: Club name
    • description: Short description
    • acceptingApplications: true or false
    • meeting: Day, time, and location
    • contacts: Leader and advisor information
  4. Save the file:
    Save the file and refresh your browser to see the changes.
⚠️ Important: Make sure to maintain proper JSON formatting. Each string must be in quotes, and don't forget commas between items!

Adding Club Images

Club images are stored in the public folder and referenced in the JSON file.

  1. Prepare your image:
    Resize your image to approximately 800x400 pixels for best results. Use JPG or PNG format.
  2. Name your image file:
    Use a simple, descriptive name like mun.jpg or eco-committee.jpg
  3. Place the image:
    Copy your image file to /public/clubs/ folder
  4. Update the JSON:
    In /data/clubs.json, update the image path:
    "image": "/clubs/your-image-name.jpg"
✓ Tip: Keep image file sizes under 500KB for faster loading. Use online tools to compress images if needed.

Example Club Entry

Here's a complete example of a club entry in the JSON file:

{ "id": "debate", "name": "Debate Club", "type": "Student-led", "category": "Academic", "image": "/clubs/debate.jpg", "description": "Develop critical thinking and public speaking skills.", "whatYouDo": [ "Weekly practice debates", "Compete in tournaments", "Learn argumentation techniques" ], "meeting": { "day": "Tuesday", "time": "15:30-17:00", "location": "Room A201" }, "contacts": { "leader": { "name": "John Smith", "email": "john.smith@student.amnuaysilpa.ac.th" }, "advisor": { "name": "Ms. Johnson", "email": "johnson@amnuaysilpa.ac.th" } }, "acceptingApplications": true, "faq": [ { "q": "Do I need experience?", "a": "No, beginners are welcome!" } ] }

Common Tasks

To close applications for a club:

Change "acceptingApplications": true to "acceptingApplications": false

To update meeting time:

Edit the values in the "meeting" object:

"meeting": { "day": "Wednesday", "time": "16:00-17:30", "location": "New Room B102" }

To add a new FAQ item:

Add a new object to the "faq" array:

"faq": [ { "q": "Existing question?", "a": "Existing answer." }, { "q": "Your new question?", "a": "Your new answer." } ]
